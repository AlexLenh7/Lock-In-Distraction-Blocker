Quick Start

Get up and running in less than a minute.

1. Enable & Add

Click the extension icon and ensure the toggle is ON. Use the "Quick Add" button when visiting a distracting site, or manually type domains (e.g., youtube.com) in the Websites tab.

2. Set Limits

Go to Settings, under Time Limit make sure it says enabled, and set your daily max time. This limit applies specifically to the websites in your block list.

3. Choose Action

Select what happens when time runs out:
Redirect to a site of your choosing
Block access completely with optional disable
Warn with a reminder in the top right corner

4. Analyze

Check the Dashboard to see your weekly usage. Insights can be viewed with the button "Learn more about your habits". Sites under 1 minute are hidden but are still counted.

Frequently Asked Questions

Why isn't a website being tracked?

Chrome restricts extensions from running on internal pages (chrome://, New Tab), the Web Store, and local files (file://). These cannot be tracked or blocked.

I added a site but it's not blocking. Why?

Ensure the toggle is ON (green icon), the Timer is set to Active in settings, and you have reloaded the tab after adding the site. Also check that you haven't dismissed a previous warning for that tab.

Does this track me when minimized?

Yes, if the window is minimized, time continues to count until the AFK/Idle detection kicks in (default is 5 minutes).

Is my data private?

100%. All data is stored locally in your Chrome browser's storage. Nothing is sent to external servers, and we do not track your usage.
